# 习概刷题网站功能升级包

本包是上一版静态刷题网站的功能升级文件。由于当前对话环境没有保留完整题库数据，本包不包含正式 `questions.js`。

## 使用方式

1. 将本包内的这些文件复制到你原来的静态网站目录：
   - `index.html`
   - `practice.html`
   - `app.js`
   - `styles.css`
   - `.nojekyll`
2. 保留你原网站里的正式 `questions.js`，并确保它和 `index.html`、`practice.html` 在同一级目录。
3. 打开 `index.html` 作为首页；点击“开始刷题”会进入独立的 `practice.html` 答题页。
4. 如果需要本地测试，可先把 `questions.js.sample` 复制一份并改名为 `questions.js`，但正式部署请使用你的原题库文件。

## 本次新增功能

- 答题界面拆成独立页面 `practice.html`。
- 练习范围支持“错题练习”，可以一次性练习全部错题。
- 错题本按章节统计，每个章节都有“练本章错题”入口。
- 返回上一题时会保留作答记录，并显示正确答案与判题结果。
- 多选题取消双击提交，避免因为双击过快导致未选完就提交。
- 单选题、判断题仍支持双击选项快速提交并进入下一题。
- 题库检索里的“练这一题”会跳转到独立答题页。

## 注意

本网站使用浏览器 `localStorage` 保存错题、收藏和练习记录。更换浏览器或清空浏览器缓存后，本机记录会消失。
